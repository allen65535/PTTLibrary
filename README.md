###### PTT Library 是一個由 Python 寫成用來操作 PTT 的函式庫，你可以在任何可以使用 Python 的地方，執行你的 PTT 機器人。
###### github: https://github.com/Truth0906/PTTLibrary
###### Pypi: https://pypi.org/project/PTTLibrary/


安裝
-------------------
```
pip3 install PTTLibrary
```

執行
-------------------
```
python bot.py
```

需求
-------------------
###### Python 3.6
